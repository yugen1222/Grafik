# Safia Schedule v7 — единая рабочая страница

## Что изменено
- Для администратора и менеджера добавлена единая страница «Рабочий график».
- Таблицы: Филиал 1 смена, Обслуживание 1 смена, Филиал 2 смена, Обслуживание 2 смена, Филиал 3 смена.
- Порядок филиала 1–2 смены: Менеджер → Кассир → Продавец → Морозильщик → Техничка.
- Порядок обслуживания 1–2 смены: Менеджер зоны → Бариста → Официант.
- Порядок 3 смены: Менеджер → Кассир → Продавец → Морозильщик → Бариста → Официант → Техничка.
- Нажатие на имя позволяет выбрать несколько выходных или всю неделю.
- Нажатие на ячейку позволяет изменить смену, категорию, вернуть из выходного или удалить назначение.
- «Скачать текущий вид» выгружает именно открытую таблицу.
- «Полный Excel» оставлен для администратора.
- Сохраняются текущие ключи localStorage и путь Firebase, поэтому существующие данные не обнуляются.
- Добавлено объединение локальных и Firebase-сотрудников, чтобы новые сотрудники не исчезали при синхронизации.

## Менеджер официантской зоны
В разделе «Навыки» добавьте менеджеру навык «Официант». Тогда он появится в таблицах «Обслуживание» и не будет дублироваться в филиальном блоке.

## Установка
Загрузите содержимое архива в папку `safia_schedule_v3` в GitHub с заменой файлов, затем выполните Commit. Render автоматически задеплоит обновление.

Перед заменой рекомендуется экспортировать JSON из Firebase Realtime Database как резервную копию.
